# 🚀 E-Commerce Product Recommendation Engine

## Overview

An industry-oriented recommendation system built using Data Structures & Algorithms, Machine Learning concepts, FastAPI, and Streamlit.

The system recommends products based on similarity analysis, ranking algorithms, and user-item interactions.

---

## Features

* Feature Store Architecture
* User & Item Aggregations
* Candidate Generation
* TF-IDF Similarity Engine
* Cosine Similarity Ranking
* Heap-Based Top-K Recommendations
* Product Similarity Graph
* Streamlit Dashboard
* FastAPI REST API
* LightGBM LambdaRank Prototype

---
